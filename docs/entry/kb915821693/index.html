Adding the Visual Studio Text Editor to Context Menus

From the Active Desktop by Internet Explorer 4.x-based systems, choose View > Folder Options > File Types > Edit... > Actions: > New... > New Action:
Action: Edit with Visual Studio
Application used to perform action:
"C:\\Program Files\\Microsoft Visual Studio            Common\\IDE\\IDE98\\devenv.exe" "%1"
Use DDE: (checked)
DDE Message: Open("%1")
Application: vstudio
DDE Application Not Running: (empty)
