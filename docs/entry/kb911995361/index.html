Microsoft Windows NT 4.0: Dial-Up Networking Settings for Netcom Unix Shell

A Dial-Up Networking Phonebook has been created and stored in my Home Directory. From my Administrator session I select Start > Programs > Accessories > Dial-Up Networking. These are the settings within Dial-Up Networking:

More > Edit entry and modem properties... Basic > Entry name: "Netcom" Basic > Use Telephony dialing properties: (not checked) Basic > Use another port if busy: (not checked) Server > Dial-up server type: "PPP:..." Server > Network Protocols > TCP/IP: (checked) Server > Network Protocols > IPX/SPX... : (not checked) Server > Network Protocols > NetBEUI: (not checked) Server > Network Protocols > TCP/IP Settings... Server assigned IP address: (selected) Specify name server addresses > Primary DNS: 199.182.203 > Secondary DNS: 199.182.202 Use IP header compression: (checked) Use default gateway on remote network: (checked) Server > Enable software compression: (checked) Server > Enable PPP LCP extensions: (not checked) Script > Pop up a terminal window
