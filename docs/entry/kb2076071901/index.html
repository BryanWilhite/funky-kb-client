tar: To unpack a standard .tar file

To unpack a standard .tar file, use this command (from the target directory):


tar -xvf [filename]



options:


x: decompress something
t: view contents
v: verbose mode
f: input from a file 



For more information, see


http://wiki.linuxquestions.org/wiki/Tar
