WindowsXP: Removing Group Policy Control of Windows Firewall; Group Policy; gpedit.msc

Run the Group Policy snap-in (gpedit.msc). Under Local Computer Policy > Computer Configuration > Administrative Templates > Network > Network Connections > Windows Firewall set "Protect all network connections" to Not Configured. Do this for Domain Profile and Standard Profile.


Run gpupdate. This program is described here:
