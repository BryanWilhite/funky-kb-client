Windows NT: Paging File Size

In System Properties, under the Performance tab, select Virtual Memory to make sure that Virtual Memory > Initial Size is set to a value that is equal to the amount of physical RAM plus 12 MB. Set Virtual Memory > Maximum Size to a value equal to or greater than the Initial Size.
