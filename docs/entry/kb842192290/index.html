Windows NT: Running the ODBC Administrator From the Command Line

Enter the command


odbcad32



in the CMD console or Run dialog to run ODBCAD32.EXE in the %SystemRoot%\\SYSTEM32 directory. This is especially useful if the current user logged in to their desktop does not have rights to the Control Panel applets.
