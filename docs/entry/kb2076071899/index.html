Ubuntu/Debian Linux: How To Change the Hostname

Edit this file:


/etc/hostname



Then run this command:


/etc/init.d/hostname.sh start



Verify that the name is changed with some of these commands:


uname -n
hostname -a
hostname -s
hostname -d
hostname -f
hostname
