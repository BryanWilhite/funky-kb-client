Windows NT Command Script: BackupProfile.cmd; Moving User Settings to a HomeDrive Other Than the SystemDrive

@ECHO OFF
ECHO Deleting old backup...
RD /S /Q "%HomeDrive%\\%HomePath%\\~data\\~Application Data Backup"
RD /S /Q "%HomeDrive%\\%HomePath%\\~data\\~Favorites Backup"
RD /S /Q "%HomeDrive%\\%HomePath%\\~data\\~Local Settings Backup"
RD /S /Q "%HomeDrive%\\%HomePath%\\~data\\~SendTo Backup"
ECHO Backing up Profile Data...
MD "%HomeDrive%\\%HomePath%\\~data\\~Application Data Backup"
MD "%HomeDrive%\\%HomePath%\\~data\\~Favorites Backup"
MD "%HomeDrive%\\%HomePath%\\~data\\~Local Settings Backup"
MD "%HomeDrive%\\%HomePath%\\~data\\~SendTo Backup"
ECHO Getting Application Data...
XCOPY "%SystemRoot%\\Profiles\\rasx\\Application Data" "%HomeDrive%\\%HomePath%\\~data\\~Application Data Backup\\" /Q /E /V
ECHO Getting Favorites...
XCOPY "%SystemRoot%\\Profiles\\rasx\\Favorites" "%HomeDrive%\\%HomePath%\\~data\\~Favorites Backup\\" /Q /E /V
ECHO Getting Local Settings...
XCOPY "%SystemRoot%\\Profiles\\rasx\\Local Settings" "%HomeDrive%\\%HomePath%\\~data\\~Local Settings Backup\\" /Q /E /V
ECHO Getting SendTo Folder...
XCOPY "%SystemRoot%\\Profiles\\rasx\\SendTo" "%HomeDrive%\\%HomePath%\\~data\\~SendTo Backup\\" /Q /E /V
