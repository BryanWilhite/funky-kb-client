Windows XP/2003 PROBLEM: Accessing Share on a Remote Machine Shows only Guest Prompt; Group Policy; gpedit.msc

Run the Group Policy snap-in (gpedit.msc). Under Local Computer Policy > Computer Configuration > Windows Settings > Security Settings > Security Options set “Network access: Sharing and security model for local accounts” to “Classic on the Windows Server 2003 family and Windows XP.”


Run gpupdate. This program is described here:
