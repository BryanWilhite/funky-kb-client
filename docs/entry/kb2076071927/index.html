SQL Server 2005 t-SQL: The Use of the modify() Method for XML Types; delete; XML Data Modification Language (DML)

DECLARE @xml xml


SET @xml =
'
<Root>
<Item1 />
<Item2 />
<Item3 />
<Item4 />
<Item5 />
</Root>
'


SET
@xml.modify
('
delete ((/Root/*[5])[1])
')


SELECT @xml


/*
<Item5 /> should be deleted.


Note that the modify() method
uses Microsoft-specific XML DML.
This is not part of the XQuery standard.

For general information, see
“Introduction to XQuery in SQL Server 2005”
http://technet.microsoft.com/en-us/library/ms345122.aspx



*/
