SharePoint Portal Server 2003: Flippant Remarks about Setup

    

  • 

    To avoid Internet Explorer security prompts, namely those popping up because of Web Parts, add the SharePoint portal to the Trusted Sites zone.
    

    • 

  • 

    The fastest way to expose the portal to an intranet domain (and all of the domains that trust this domain) is to add the NT AUTHORITY\\Authenticated Users group to the SharePoint Readers group.
