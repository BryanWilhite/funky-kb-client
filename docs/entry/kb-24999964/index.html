Netcom Shell: Setting up a PPP Session on a Power Macintosh; System 7.x

    

  1. 

    Install Open Transport (OT 1.1.2). You may want to turn off virus detection during installation.
    

    2. 

  2. 

    Install Open Transport PPP (1.0)
    

    3. 

  3. 

    Run Control Panel TCP/IP with the following settings: Connect Via = PPP; Configure = Manually; IP Address/ Name Server Domains = 199.182.120.203
    

    4. 

  4. 

    Run Control Panel PPP: enter name and password; Number = 9,1-310-815-4000; Options... Protocol = Connect to a command-line host = Use terminal window.
    

    5. 

  5. 

    Run modem control panel and choose modem.
