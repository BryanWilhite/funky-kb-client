MS SQL Server 6.5: ISQL/w Can Connect to Sybase Databases

ISQL/w can read both Microsoft and Sybase SQL server databases. The SQL Client Configuration Utility can be used to add a Sybase SQL Server to the list of available servers. Under the Advanced > Client Configuration, type in a Sybase server name and select TCP/IP Sockets for DLL Name. Enter the server name, a comma and the port number the Sybase server answers.
