ASP.NET ERROR in Visual Studio: "Unable to start debugging on the web server. Debugging failed because integrated Windows authentication is not enabled. Please see Help for assistance."

Windows XP Solution: In the Internet Information Services (IIS) Management Console under Default Web Site Properties > Directory Security > Anonymous Access and Authentication Control > Edit... > Authenticated Access > Integrated Windows authentication, check the check box.
