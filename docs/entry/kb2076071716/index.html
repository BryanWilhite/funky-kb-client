.htaccess WARNING for Apache Servers: Do not encode the .htaccess file in UTF-8

Do not encode the .htaccess file in UTF-8. This will cause an internal server error. Just because you can do this:


AddDefaultCharset UTF-8



does not mean that the .htaccess file itself should be encoded as UTF-8. For more information, see "Various .htaccess samples and tutorials" here:
