ASP: Denying Anonymous Access with the ServerVariables Dictionary Object

'Deny anonymous access.
If Request.ServerVariables("LOGON_USER") = Empty Then
Response.Status = "401 Access Denied"
End If


'For the converse of this issue please see MS KB article Q188717
'"PRB: Request.ServerVariables("LOGON_USER") Returns Empty String"
