Dexpot 1.4 PROBLEM: Switching from Desktop to Desktop Leaves ‘Junk’ on the Screen in Windows XP

Switching from desktop to desktop in Dexpot 1.4 leaves abandoned border artifacts on the screen. This comes from its lack of support for “shadow effects” in Windows XP. To resolve this annoyance, check off “Show shadows under menus” Display Properties > Effects… in  to disable this effect.
