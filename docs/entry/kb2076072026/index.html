Microsoft Word 2007: How to change the default view to Draft view in Word 2007; support.microsoft.com

To change the default view to Draft view when a Word 2007 document is opened, follow these steps:


    

  1. 

    Click the Microsoft Office Button, and then click Word Options.
    

    2. 

  2. 

    Click Advanced.
    

    3. 

  3. 

    Under General, click to select the Allow opening a document in Draft view check box.
    

    4. 

  4. 

    Click OK.
    

    5. 

  5. 

    On the View tab, click Draft in the Document Views group.
    

    6.
