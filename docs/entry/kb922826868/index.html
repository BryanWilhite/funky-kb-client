The vbYesNoCancel Form with the VBA.MsgBox Statement

'The following design should speak for itself:


Select Case VBA.MsgBox(&quot;Yes, No or Cancel?&quot;, _
    vbQuestion + vbYesNoCancel)
    Case vbYes
        'The Yes case.
    Case vbNo
        'The No case.
    Case vbCancel
        'The Cancel case.
End Select
