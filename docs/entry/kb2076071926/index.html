SQL Server 2005 t-SQL: The Use of the modify() Method for XML Types; insert element; XML Data Modification Language (DML)

DECLARE @xml xml


SET @xml =
'
<Root>
<Item1 />
<Item2 />
<Item3 />
<Item4 />
<Item5 />
</Root>
'


SET
@xml.modify
('
insert element Item6 {} as last into (/Root)[1]
')


SELECT @xml


/*
This projection produces (note the empty brackets above, {},
indicating a nil value for Item6):


&lt;Root&gt;
  &lt;Item1 /&gt;
  &lt;Item2 /&gt;
  &lt;Item3 /&gt;
  &lt;Item4 /&gt;
  &lt;Item5 /&gt;
  &lt;Item6 /&gt;
&lt;/Root&gt;

Note that the modify() method
uses Microsoft-specific XML DML.
This is not part of the XQuery standard.

For general information, see
“Introduction to XQuery in SQL Server 2005”
http://technet.microsoft.com/en-us/library/ms345122.aspx



*/
