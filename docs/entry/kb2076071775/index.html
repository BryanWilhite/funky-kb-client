eclipse 3.x SDK: Filtering TODO Tasks Per Package

Filtering TODO tasks per package means filtering "On selected resource and its children." In the Tasks View (Window > Show View > Tasks), click the Filters... button to show the Filters dialog. Select the "On selected resource and its children" option to filter Tasks per package---or any Resource with child items.
