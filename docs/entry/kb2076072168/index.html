SQLite: Recommended Management Tools for Ubuntu Linux; GUIs

>>>Sqliteman http://sqliteman.com/ OpenSUSE binaries available from the site. A Debian package (that depends on Qt Toolkit) is available here:

http://froebe.net/blog/2008/04/16/sqliteman-ubuntu-i386-package/

Direct download:

http://www.froebe.net/packages/sqliteman_1.0.1-1_i386.deb

>>>SQLiteStudio http://sqlitestudio.one.pl This one just works as a binary packed with its own GTK libraries.

>>>sqlite-manager (FireFox Plugin) http://code.google.com/p/sqlite-manager/wiki/SupportedApplications Works well in FireFox 3.5 and above. Also, a XULRunner option is available. # Mozilla Addons site: https://addons.mozilla.org/en-US/firefox/addon/5817

Full list of GUIs: http://www.sqlite.org/cvstrac/wiki?p=ManagementTools
