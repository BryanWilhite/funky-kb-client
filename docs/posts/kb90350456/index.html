Yes, style sheets have the padding-top and margin-top attributes, but if there is a concern about a browser (like Netscape Navigator 3.x) that may not support style sheets, try using the non-breaking space entity (Â ). To force line spacing above data in a table cell, try this:

&lt;TABLE&gt;
&lt;TR&gt;
    &lt;TD&gt;This text is in row one of this table.&lt;/TD&gt;
&lt;/TR&gt;
&lt;TR&gt;
    &lt;TD&gt;&nbsp;&lt;BR&gt;&lt;BR&gt;This text is in row two.&lt;/TD&gt;
&lt;/TR&gt;
&lt;/TABLE&gt;

where the text in the second row of the table has a little more "padding" on the top thanks to the non-breaking space. The second TD block has a "hidden" line of text consisting of the single non-breaking space character. The <BR> tags then "break" this hidden line.
