When the "power" end user makes many custom changes to Office Applications, it is possible to save these changes using REGEDIT.EXE (not REGEDT32.EXE for Windows NT users) in a .REG file. For Microsoft Access 97, export this key:

HKEY_CURRENT_USER
    &bsol;&bsol;Software&bsol;&bsol;Microsoft&bsol;&bsol;Office&bsol;&bsol;8.0&bsol;&bsol;Access&bsol;&bsol;Settings

For Microsoft Word 97, export these keys:

HKEY_CURRENT_USER
    &bsol;&bsol;Software&bsol;&bsol;Microsoft&bsol;&bsol;Office&bsol;&bsol;8.0&bsol;&bsol;Word&bsol;&bsol;Data

HKEY_CURRENT_USER
    &bsol;&bsol;Software&bsol;&bsol;Microsoft&bsol;&bsol;Office&bsol;&bsol;8.0&bsol;&bsol;Word&bsol;&bsol;Options

For more information please see the Microsoft Knowledge Base, "WD: How to Rebuild User Preference and Options Settings" (Q140097).
