Create a new folder in your Start Menu. To be able to do this you can run a command like:

C:&bsol;&bsol;WINNT&bsol;&bsol;Profiles&bsol;&bsol;rasx&bsol;&bsol;Start Menu

if your system root directory is C:\\WINNT and your user name is "rasx." Create a new folder inside the Start Menu folder. Copy the following string:

Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}

and paste it over the "New Folder" default name during the Rename mode---which appears by default when you create a new folder.
