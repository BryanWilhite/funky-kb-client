One glaring difference in the CREATE TABLE syntax between SQL 2000 and 2005 is this 2005 clause:

    WITH (IGNORE_DUP_KEY = OFF) ON [PRIMARY]

The only WITH clause support in SQL Server 2000 is WITH FILLFACTOR.
